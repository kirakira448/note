思路：使用`v-if` 实现

## 示例：
路由
```js
{
  path:'trouble',
  name:'trouble',
  component:() => import('@/views/Management/Vehicle/Trouble/VehicleTrouble.vue'),
  children:[
	{
	  path:'detail',
	  name:'troubleDetail',
	  component:() => import('@/views/Management/Vehicle/Trouble/TroubleDetail.vue'),
	},

  ]
},
```

组件
```vue
<template>
    <MainLayout v-if="$router.currentRoute.value['fullPath']==='/vehicle/trouble'">
	</MainLayout>

    <RouterView v-else/>
</template>
```
