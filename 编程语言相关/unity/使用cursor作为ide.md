## Unity端

### 1. 添加插件
- Unity->Window->Package Manager  
- Click "+" left corner  
- Add package from git URL  
- Insert `https://github.com/boxqkrtm/com.unity.ide.cursor.git`  
- Add  
- Done

### 2. 设置使用cursor

- `Edit` -> `Preferences..` -> `External Tools`
- 选择 Cursor
- 点击按钮 `Regenerate project files`
