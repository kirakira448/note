
## 描述

不同语言环境下生成的文件可能会导致乱码。
例如，中文环境下生成的文件在其他语言环境下出现乱码：
![[Snipaste_2024-11-11_11-22-13.png]]


## 解决方法

### 1. 安装`7-zip.exe`

https://www.7-zip.org/

### 2. 使用命令行解压

1) 复制`7zG.exe`路径

	打开安装路径，复制`7zG.exe`
2) 复制`目标文件.zip`路径
3) 整理最终命令
```bash
7zG.exe x 压缩包.zip -mcp=936
```
