
## 1. 创建基本项目结构
- `index.html`
- `main.js`
- `assets/`：静态文件存储目录

`index.html`
```html
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="utf-8">
		<title>My first three.js app</title>
		<style>
			body { margin: 0; }
		</style>
	</head>
	<body>
		<script type="module" src="/main.js"></script>
	</body>
</html>
```

`main.js`
```js
import * as THREE from 'three';
```


## 2. 安装
### 2.1 安装`Node.js`
### 2.2 安装`three.js` 和 Vite

`three.js`
```shell
npm install --save three
```

`vite`
```shell
npm install --save-dev vite
```

### 2.3 在终端运行

```shell
npx vite
```
