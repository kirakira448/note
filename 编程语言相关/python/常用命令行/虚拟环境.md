## Anaconda
### 创建

```shell
conda create -n env_name python=2.7
```
不指定python版本时，自动安装最新python版本。

### 激活

```shell
conda activate your_env_name
```

### 退出

```shell
deactivate env_name
```
或者激活root
```shell
activate root
```


### 删除

```shell
conda remove -n your_env_name(虚拟环境名称) --all
```
